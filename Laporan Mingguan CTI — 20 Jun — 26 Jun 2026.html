MITRE: T1219 — Remote Access Software | Tactic: Command and Control
Kampanye malware AsyncRAT (Win.AsyncRAT) merupakan ancaman berbasis Remote Access Trojan (RAT) yang memberikan akses penuh kepada penyerang untuk mengendalikan sistem korban, mencuri kredensial, merekam aktivitas pengguna, melakukan keylogging, mengunduh malware tambahan, hingga menjadi pintu masuk ransomware. Kampanye AsyncRAT umumnya disebarkan melalui phishing email, lampiran berbahaya, file ZIP/LNK, penyalahgunaan layanan cloud seperti Dropbox dan Cloudflare, serta teknik social engineering modern seperti ClickFix. Berdasarkan berbagai laporan intelijen ancaman, sektor yang paling sering menjadi target meliputi keuangan, pemerintahan, kesehatan, teknologi, manufaktur, hospitality/travel, aerospace, pendidikan, serta critical infrastructure. Aktivitas kampanye bersifat global dengan fokus pada pencurian kredensial, akses awal (initial access), pengintaian jaringan internal, dan eksfiltrasi data sensitif. Organisasi disarankan untuk meningkatkan kontrol keamanan email dan web filtering guna memblokir lampiran serta tautan berbahaya, menerapkan Multi-Factor Authentication (MFA) pada seluruh akun kritikal, memantau koneksi outbound ke domain atau IP Command & Control (C2) yang tidak dikenal, serta melakukan threat hunting terhadap indikator seperti eksekusi PowerShell, mshta.exe, rundll32.exe, regsvr32.exe, scheduled task mencurigakan, dan proses yang melakukan koneksi jaringan tidak lazim. Selain itu, perlu dilakukan pemantauan aktivitas pencurian kredensial, penerapan Endpoint Detection and Response (EDR), pembaruan sistem secara berkala, serta peningkatan kesadaran pengguna terhadap teknik phishing dan social engineering yang sering digunakan dalam distribusi AsyncRAT

MITRE: T1219 — Remote Access Software | Tactic: Command and Control
Kampanye Mirai (ELF.Mirai) merupakan malware botnet yang secara khusus menargetkan sistem berbasis Linux dan perangkat Internet of Things (IoT) seperti router, DVR, IP Camera, NAS, dan perangkat jaringan lainnya. Mirai bekerja dengan melakukan pemindaian internet secara masif untuk mencari perangkat yang menggunakan kredensial default atau lemah, kemudian menginfeksi perangkat tersebut dan menghubungkannya ke infrastruktur Command & Control (C2). Setelah terinfeksi, perangkat akan menjadi bagian dari botnet yang dapat digunakan untuk melancarkan serangan Distributed Denial of Service (DDoS) berskala besar, penyebaran malware lanjutan, maupun aktivitas berbahaya lainnya. Sektor yang paling berisiko terdampak meliputi telekomunikasi, penyedia layanan internet (ISP), teknologi, pemerintahan, pendidikan, kesehatan, manufaktur, serta organisasi yang mengoperasikan perangkat IoT dan OT/ICS dalam jumlah besar. Organisasi disarankan untuk melakukan inventarisasi seluruh perangkat IoT dan Linux yang terhubung ke jaringan, mengganti kredensial bawaan (default credentials), menerapkan password yang kuat, serta memastikan firmware dan sistem operasi selalu diperbarui. Selain itu, perlu dilakukan monitoring terhadap aktivitas scanning outbound maupun inbound yang tidak normal, koneksi menuju Command & Control (C2), serta lonjakan trafik jaringan yang dapat mengindikasikan aktivitas botnet atau DDoS. Segmentasi jaringan untuk perangkat IoT, pembatasan akses manajemen dari internet, penerapan firewall dan IDS/IPS, serta terhadap proses dan koneksi mencurigakan pada perangkat Linux menjadi langkah penting untuk mengurangi risiko kompromi dan mencegah perangkat menjadi bagian dari infrastruktur botnet Mirai.

MITRE: T1219 — Remote Access Software | Tactic: Command and Control
Kampanye Adaptix C2 (Win.Adaptix_C2) merupakan ancaman berbasis Command and Control (C2) framework yang digunakan oleh pelaku ancaman untuk mempertahankan akses ke sistem yang telah berhasil dikompromi. Framework ini memungkinkan penyerang menjalankan perintah jarak jauh, melakukan enumerasi sistem, mengumpulkan kredensial, melakukan lateral movement, serta mengunduh payload tambahan seperti infostealer, RAT, atau ransomware. Adaptix C2 sering muncul pada tahap post-exploitation setelah keberhasilan phishing, eksploitasi kerentanan publik, atau penyalahgunaan kredensial yang valid. Berdasarkan pola penggunaan C2 framework modern, target serangan umumnya meliputi sektor pemerintahan, keuangan, teknologi, manufaktur, kesehatan, pendidikan, dan critical infrastructure, dengan tujuan memperoleh akses jangka panjang, pencurian data sensitif, dan persiapan serangan lanjutan. Organisasi disarankan untuk melakukan pemantauan aktif terhadap koneksi outbound yang tidak biasa, khususnya komunikasi terenkripsi menuju domain atau IP yang tidak dikenal, melakukan terhadap proses yang melakukan network beaconing secara periodik, serta memonitor eksekusi tool pascakompromi seperti PowerShell, cmd.exe, rundll32.exe, regsvr32.exe, mshta.exe, dan LOLBins lainnya. Selain itu, penerapan Multi-Factor Authentication (MFA), segmentasi jaringan, pembatasan hak akses administratif, penerapan Endpoint Detection and Response (EDR), serta monitoring aktivitas autentikasi yang mencurigakan perlu diperkuat guna mendeteksi dan memutus komunikasi Command & Control sebelum penyerang dapat melakukan eskalasi serangan atau eksfiltrasi data.

MITRE: T1204.002 — User Execution: Malicious File | Tactic: Execution
Kampanye 32-bit merujuk pada jenis malware dropper atau loader yang dikompilasi secara spesifik untuk arsitektur sistem 32-bit, yang sering digunakan sebagai tahap awal dalam rantai serangan siber. Malware ini berfungsi sebagai pintu masuk untuk mengunduh dan mengeksekusi payload berbahaya tambahan ke dalam sistem target. Teknik penyebarannya sangat bergantung pada manipulasi psikologis (rekayasa sosial), di mana pengguna diarahkan untuk membuka atau mengunduh file berbahaya melalui tautan yang disisipkan dalam pesan phishing atau situs web yang telah disusupi, yang sesuai dengan taktik MITRE ATT&CK T1204.002 — User Execution: Malicious File. Setelah dieksekusi, malware ini akan menghubungi server eksternal untuk mengunduh komponen berbahaya lainnya, menciptakan persistensi pada sistem, dan mempersiapkan lingkungan bagi ancaman yang lebih destruktif. Sektor yang sangat rentan terhadap kampanye ini meliputi sektor operasional yang masih mengandalkan sistem legacy (warisan) 32-bit, administrasi perkantoran, dan industri yang sering berinteraksi dengan unduhan dokumen dari luar jaringan internal.

MITRE: T1566.002 — Phishing: Spearphishing Link | Tactic: Initial Access
Kampanye ELF.Mozi merupakan kampanye malware botnet yang menargetkan perangkat Internet of Things (IoT) berbasis Linux, seperti router, kamera IP, DVR, dan perangkat jaringan lainnya yang memiliki layanan manajemen terbuka atau menggunakan kredensial bawaan yang lemah. Malware ini berfungsi untuk mengambil alih perangkat yang rentan dan memasukkannya ke dalam botnet yang dikendalikan melalui mekanisme peer-to-peer (P2P) berbasis Distributed Hash Table (DHT), sehingga tidak bergantung pada satu server Command and Control (C2). Teknik penyebarannya memanfaatkan eksploitasi layanan yang terekspos ke internet, seperti Telnet dan SSH, serta pemanfaatan kredensial yang lemah atau default credentials, yang sesuai dengan taktik MITRE ATT&CK T1110 Brute Force (Brute Force), T1021 Remote Services (Remote Services), dan T1190 Exploit Public-Facing Application (Exploit Public-Facing Application). Setelah berhasil menginfeksi perangkat, Mozi akan mempertahankan akses, berkomunikasi dengan node botnet lain melalui jaringan P2P, serta dapat menerima perintah untuk melakukan serangan Distributed Denial-of-Service (DDoS), penyebaran malware lebih lanjut, maupun aktivitas berbahaya lainnya. Sektor yang paling rentan terhadap kampanye ini meliputi penyedia layanan internet (ISP), telekomunikasi, pemerintahan, manufaktur, pendidikan, kesehatan, perusahaan yang mengelola infrastruktur jaringan skala besar, serta organisasi maupun pengguna rumahan yang masih menggunakan perangkat IoT dengan konfigurasi keamanan yang lemah atau belum diperbarui.

MITRE: T1566.002 — Phishing: Spearphishing Link | Tactic: Initial Access
Kampanye ELF.BackdoorIT merujuk pada aktivitas penyebaran malware backdoor yang menargetkan sistem operasi Linux berbasis format ELF (Executable and Linkable Format). Malware ini dirancang untuk memberikan akses jarak jauh secara tidak sah kepada penyerang, sehingga memungkinkan pelaksanaan perintah jarak jauh (remote command execution), pengunduhan dan eksekusi payload tambahan, pencurian informasi, serta pengendalian penuh terhadap sistem yang telah terinfeksi. Teknik penyebarannya umumnya memanfaatkan eksploitasi layanan yang terekspos ke internet, kredensial yang lemah atau dicuri pada layanan SSH, serta eksploitasi kerentanan pada aplikasi atau layanan publik, yang sesuai dengan taktik MITRE ATT&CK T1190 Exploit Public-Facing Application, T1110 Brute Force, dan T1021.004 Remote Services: SSH. Setelah berhasil dijalankan, malware akan membangun persistensi, membuka saluran komunikasi dengan server Command and Control (C2), menerima instruksi dari penyerang, serta mengunduh atau menjalankan malware tambahan sesuai kebutuhan operasi, yang berkaitan dengan teknik T1105 Ingress Tool Transfer, T1071 Application Layer Protocol, dan T1053 Scheduled Task/Job apabila digunakan untuk mempertahankan persistensi. Sektor yang sangat rentan terhadap kampanye ini meliputi penyedia layanan cloud, pusat data, telekomunikasi, penyedia layanan internet (ISP), perusahaan teknologi, pemerintahan, institusi pendidikan, serta organisasi yang mengoperasikan server Linux yang terhubung ke internet, terutama apabila layanan SSH, aplikasi web, atau sistem administrasi jarak jauh tidak dikonfigurasi dengan baik atau belum mendapatkan pembaruan keamanan terbaru.

MITRE: T1204.002 — User Execution: Malicious File | Tactic: Execution
Kampanye Win.FormBook merupakan ancaman berbasis information stealer dan malware-as-a-service (MaaS) yang menargetkan sistem operasi Windows untuk mencuri informasi sensitif pengguna. Malware ini mampu merekam penekanan tombol (keylogging), mencuri kredensial yang tersimpan pada peramban web dan aplikasi klien email, mengambil isi clipboard, serta mengekstraksi informasi sistem sebelum mengirimkannya ke server Command and Control (C2). Teknik penyebarannya sangat bergantung pada rekayasa sosial melalui kampanye phishing yang mengirimkan lampiran atau tautan berbahaya, yang sesuai dengan taktik MITRE ATT&CK T1566 Phishing dan T1204.002 User Execution: Malicious File. Setelah dieksekusi oleh korban, malware akan membangun persistensi pada sistem, mengumpulkan kredensial pengguna, melakukan keylogging, serta berkomunikasi dengan server C2 melalui protokol HTTP atau HTTPS untuk mengirimkan data hasil pencurian, yang berkaitan dengan teknik T1056.001 Keylogging, T1555 Credentials from Password Stores, dan T1071.001 Application Layer Protocol: Web Protocols. Sektor yang sangat rentan terhadap kampanye ini meliputi perbankan dan layanan keuangan, pemerintahan, pendidikan, manufaktur, kesehatan, perusahaan jasa, serta organisasi yang mengandalkan komunikasi email dan aktivitas berbasis web dalam operasional sehari-hari, karena tingginya risiko pengguna membuka lampiran atau dokumen dari sumber eksternal.
